## 执行计划

### 阶段 1: 单元测试验证
```bash
cd backend && uv run pytest tests/unit/training/test_svc_*.py -v
```

### 阶段 2: 集成测试验证
```bash
cd backend && uv run pytest tests/integration/training/ -v
```

### 阶段 3: E2E 验证 (需真实集群)
```bash
cd backend && HYPERPOD_ENABLE_WRITE_TESTS=true uv run pytest tests/e2e/scenarios/ -v
```

### 阶段 4: 覆盖率报告
```bash
cd backend && uv run pytest tests/unit/training/ --cov=src/modules/training --cov-report=html
```

## 下一步行动

1. **执行评估**: 运行 `/eval check hyperpod-integration-service` 检查各评估项状态
2. **修复问题**: 根据评估结果修复未通过的测试
3. **生成报告**: 运行 `/eval report hyperpod-integration-service` 生成完整评估报告
