# HyperPod 训练任务提交评估定义

## 概述

为基于 AWS SageMaker HyperPod Task Governance API 的训练任务提交功能创建评估定义文件。

## 已完成

✅ 创建评估定义文件: `.claude/evals/hyperpod-training-job-submission.md`

## 评估定义内容

### 能力评估 (8 个类别, 28 项检查)

| 类别 | 检查项数 | 覆盖范围 |
|------|---------|---------|
| CE-01: 基础任务提交 | 4 | API 创建、状态初始化、HyperPod 提交、数据库记录 |
| CE-02: 分布式训练配置 | 5 | DDP/FSDP/DeepSpeed、多节点、每节点任务数 |
| CE-03: 优先级调度 | 4 | 高/中/低优先级、Kueue PriorityClass |
| CE-04: 检查点配置 | 3 | 间隔配置、恢复配置、存储路径 |
| CE-05: 状态同步 | 3 | HyperPod→DB 同步、状态映射、特殊状态处理 |
| CE-06: 身份认证与授权 | 4 | 401/403 验证、角色检查、owner_id |
| CE-07: 输入验证 | 4 | 必填字段、枚举验证、唯一性、格式验证 |
| CE-08: HyperPod SDK 集成 | 4 | 集群上下文、任务提交、配置转换、异常处理 |

### 回归评估 (5 个类别, 13 项检查)

| 类别 | 检查项数 | 覆盖范围 |
|------|---------|---------|
| RE-01: API 兼容性 | 3 | 列表、详情、筛选排序 |
| RE-02: 认证系统 | 3 | JWT、角色、Token 过期 |
| RE-03: 数据库操作 | 3 | CRUD、事务、并发 |
| RE-04: 错误处理 | 3 | 异常映射、RFC 7807、安全 |
| RE-05: 日志与可观测性 | 3 | 审计、追踪、错误日志 |

### 成功标准

- Capability pass@3 > 90%
- Regression pass^3 = 100%
- Unit Test Coverage ≥ 80%
- Integration Test 100% Pass
- E2E 至少 1 个真实 HyperPod 提交成功

## 后续操作

使用以下命令运行评估:
- `/eval check hyperpod-training-job-submission` - 运行评估检查
- `/eval report hyperpod-training-job-submission` - 生成完整报告
