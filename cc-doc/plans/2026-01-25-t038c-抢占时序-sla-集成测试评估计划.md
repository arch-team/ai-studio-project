**Pod 状态轮询**:
```python
async def wait_for_pods_terminated(job_id: str, timeout: int = 60) -> datetime:
    start = datetime.utcnow()
    while (datetime.utcnow() - start).total_seconds() < timeout:
        pods = await list_pods_for_job(job_id)
        if not pods:
            return datetime.utcnow()
        await asyncio.sleep(1)
    raise TimeoutError("Pods not terminated within timeout")
```

- 测试后清理: 删除创建的所有资源
