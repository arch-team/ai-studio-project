# 更新 frontend 文档体系：5 个文件与项目实际对齐

## 背景

`frontend/docs/context-guide.md` 和 `frontend/docs/project-config.template.md` 需要与项目实际情况同步。同时用户新增的 3 个 `.claude` 文件 (`rules/tech-stack.md`, `rules/project-structure.md`, `project-config.md`) 内容来自通用模板，与项目实际严重不符，需一并修正。

## 项目实际情况速查（修正依据）

**package.json 名称**: `ai-training-platform-frontend`
**包管理器**: npm (非 pnpm)
**CSS**: AWS Cloudscape（无 TailwindCSS）
**ESLint**: `.eslintrc.cjs` (传统格式，非 flat config)
**Prettier**: 不存在
**Features 模块 (12个)**: training, datasets, models, spaces, audit, billing, monitoring, templates, reports, admin, auth, resource-quotas
