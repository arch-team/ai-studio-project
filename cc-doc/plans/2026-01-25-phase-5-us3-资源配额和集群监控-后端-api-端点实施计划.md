## API 端点规格详情

### T055: GET /users

```yaml
Path: /users
Method: GET
Auth: require_admin
Parameters:
  - page: int (default=1)
  - page_size: int (default=20, max=100)
  - role: enum [admin, user, viewer] (optional)
  - status: enum [active, inactive, suspended] (optional)
  - sort_by: string (default="created_at")
  - sort_order: enum [asc, desc] (default=desc)
Response 200:
  {
    "items": [UserResponse],
    "total": int,
    "page": int,
    "page_size": int,
    "total_pages": int
  }
Response 403: Forbidden (非管理员)
```

### T056: POST /users

```yaml
Path: /users
Method: POST
Auth: require_admin
Request Body:
  {
    "username": string (required),
    "email": string (required, email format),
    "display_name": string (optional),
    "role": enum [admin, user, viewer] (required),
    "resource_quota_id": int (optional)
  }
Response 201:
  UserResponse
Response 409: Conflict (用户名或邮箱已存在)
Response 422: Validation Error
```

## 风险和注意事项

1. **Prometheus 端点配置**: 需要从环境变量获取 Prometheus endpoint URL
2. **权限检查**: 所有用户管理端点必须使用 `require_admin` 装饰器
3. **分页一致性**: 使用 `shared/api/pagination` 模块确保分页响应格式统一
4. **异常处理**: 使用 `shared/domain/exceptions` 中定义的异常类
5. **缓存策略**: T220 训练指标查询需要实现缓存（已完成任务缓存 1 小时）
